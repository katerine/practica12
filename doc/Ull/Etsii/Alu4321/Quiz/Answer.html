Recibe como parC!metro una lista de dos elementos que contiene el state o estado de la respuesta (si es verdadera o falsa) y el value con el texto que la representa.

# File ull-etsii-alu4321-quiz/lib/ull-etsii-alu4321-quiz.rb, line 154
def initialize(ans)
    raise ArgumentError, "Array spected, got #{ans.class}" unless ans.is_a? Array
    raise IndexError, 'Must have two (2) elements; state and value' unless ans.size == 2
    state = ans[0]
    value = ans[1]
    state == :right ? @state = RIGHT : @state = WRONG
    @value = value
end
